# CareerMatch AI

## Overview
CareerMatch AI is an NLP-based Resume Analyzer that compares a candidate's resume with a job description and calculates a match score.

## Features
- Resume PDF Upload
- Job Description Input
- Match Score Calculation
- Skill Extraction
- Missing Skill Detection

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## How It Works
1. Extracts text from resume
2. Converts text into TF-IDF vectors
3. Computes similarity score
4. Detects skill gaps
